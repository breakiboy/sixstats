Données de Performance Actuelles

Nous sommes heureux de partager nos statistiques du serveur avec vous. Veuillez noter que les données quotidiennes peuvent avoir un retard de 48 à 72 heures. Par conséquent, les données les plus récentes peuvent manquer dans le graphique quotidien car elles sont incomplètes.

Nous utilisons exclusivement Google Analytics pour analyser nos données, vous offrant ainsi les informations les plus précises. Les données en direct montrent les activités des 30 dernières minutes, y compris les utilisateurs et leurs pays. Ces données sont mises à jour toutes les 5 minutes.

Pour permettre un décompte détaillé, vous pouvez cliquer sur les graphiques. Les données statistiques sont 100 % anonymisées.
